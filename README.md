# Biznes hüququ - 3 hissəli imtahan sistemi

Bu sayt Biznes hüququ fənni üçün hazırlanmış 3 hissəli özünü yoxlama imtahan sistemidir.

## Hissələr

- I hissə: 1-100-cü suallar
- II hissə: 101-200-cü suallar
- III hissə: 201-300-cü suallar

## Funksiyalar

- Hər hissə dashboardda ayrıca göstərilir
- Hər hissə üçün ayrıca giriş/başlama imkanı
- Hər hissədə 100 sualın hamısı təqdim olunur
- Hər imtahan müddəti: 90 dəqiqə
- Suallar hər girişdə qarışdırılır
- Cavab variantlarının yeri hər girişdə dəyişir
- Düzgün cavablar gizli saxlanılır
- Sağ tərəfdə sual naviqasiyası var
- Cavablanan suallar mavi, baxılıb cavabsız qalan suallar sarı göstərilir
- Sonda düzgün/səhv/cavabsız cavablar göstərilir
- Hər hissə üçün ayrıca lokal reytinq cədvəli var

## Sual sayı

Excel faylından oxunan ümumi sual sayı: 300
