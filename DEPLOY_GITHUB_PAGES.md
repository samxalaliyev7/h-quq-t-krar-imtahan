# GitHub Pages yerləşdirmə qaydası

1. ZIP faylını kompüterdə açın.
2. İçindəki faylları GitHub repository-yə yükləyin.
3. `Commit changes` düyməsini basın.
4. `Settings` bölməsinə daxil olun.
5. `Pages` bölməsini açın.
6. Source: `Deploy from a branch`
7. Branch: `main`
8. Folder: `/root`
9. `Save` düyməsini basın.
10. 2-5 dəqiqə gözləyin.

Link formatı:

https://samxalaliyev7.github.io/REPOSITORY_ADI/

Cache üçün yoxlama linkinin sonuna belə versiya əlavə edə bilərsiniz:

?v=biznes-huququ-3-part-v1
